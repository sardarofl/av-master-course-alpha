LED Controller vs Processor, Canvas Concepts & Hybrid Processors

Understanding LED Display Systems

LED walls are fundamentally different from LCD displays. They use thousands of individual LED modules that must be controlled by a sophisticated processing chain. This module covers the LED processing ecosystem - from video source to emitted light - including the critical distinction between controllers and processors, how canvas size works, and when to use hybrid solutions.

  • The difference between LED controllers and LED processors
  • How canvas size and virtual canvas work
  • The role of receiving cards and sending cards
  • Major processor brands: NovaStar vs Brompton
  • Hybrid processor capabilities
  • How to configure an LED processing chain

LED Controller vs LED Processor: Key Differences

These terms are often confused, but they serve different purposes in the signal chain.

🎛️ LED Processor

Role: Signal enhancement and manipulation

Inputs: HDMI, SDI, DVI, IP

Outputs: Video data to controller

The "brain" that enhances video quality. Handles scaling, color correction, image effects, PIP/POP, and prepares the signal for the LED wall.

Functions: Scaling, color calibration, HDR, frame rate conversion, multi-source layouts

🔌 LED Controller

Role: Panel management and distribution

Input: Processed video data

Outputs: Ethernet/Cat6 to panels

The "distribution hub" for panels. Takes processed video and maps it to the correct positions on each LED module.

Functions: Panel mapping, loading, receiving card management, redundancy

⚠️ Essential Distinction

You need BOTH for a complete LED system:

  • Processor alone: Creates enhanced video but can't drive panels directly
  • Controller alone: Can drive panels but without advanced processing
  • Together: Complete signal chain from source to emitted light

Many modern systems combine both functions into a single "processor/controller" or "hybrid" device.

Canvas Size and Virtual Canvas

The canvas is the total resolution area that the LED wall displays. Understanding canvas vs. output resolution is critical for proper configuration.

📐 Key Canvas Concepts
Virtual Canvas Size: The total resolution area your content spans (e.g., 1920×1080 for a single 16:9 area).
Output Resolution: The actual pixel count of your LED panel array (e.g., 3840×1920 for a 2x2 array of 1920×960 panels).
The processor must scale/crop the virtual canvas to match the output resolution.

Major LED Processor Brands

⭐ NovaStar

Market: Entry to mid-level

Strength: Cost-effective, widely compatible

Features: COEX series, NovaLCT software

The most common choice for general LED installations. Good balance of features and price.

Best For: Fixed installations, rental staging, cost-conscious projects

👑 Brompton

Market: Professional/High-end

Strength: Gold standard for live events

Features: PureTone, DarkMagic, Tessera SX40

The industry benchmark for premium processing. Superior color accuracy and feature set.

Best For: Broadcast, film production, high-profile live events

🔧 Colorlight

Market: Budget/Entry-level

Strength: Lowest cost option

Features: Basic processing, LEDVISION software

Entry-level choice for simple LED installations. Limited advanced features.

Hybrid Processors: All-in-One Solutions

Modern LED systems often use hybrid processors that combine controller and processor functions in a single unit.

✅ Advantages of Hybrid

  • Single device to purchase and configure
  • Lower total cost for simple installations
  • Simpler signal chain, fewer points of failure
  • Compact form factor
  • Integrated control software

⚠️ Limitations of Hybrid

  • Limited input count (typically 2-4 inputs)
  • Fixed output capacity (can't scale to massive walls)
  • Fewer advanced features than separate systems
  • Less redundancy options
  • Not suitable for broadcast/camera work

Lab Exercise: Configure LED Processing Chain

Objective: Configure an LED processor and controller for a 3×3 LED wall display.

1
Gather Equipment
  • 3×3 LED wall (9 panels, 640×360 each = 1920×1080 total)
  • LED processor with scaling and color management
  • LED controller with sending card
  • Video source (4K laptop)
  • Ethernet cables for panel connections
  • Processor software (NovaLCT, LEDVision, or similar)
2
Physical Setup
  • Mount LED panels in 3×3 grid
  • Connect panels to controller via Ethernet
  • Connect controller to processor output
  • Connect video source to processor input
3
Software Configuration
  • Define panel configuration: 3×3 grid
  • Set canvas resolution: Match source or desired output
  • Configure output resolution: 1920×1080
  • Map panels to correct positions
  • Set bezel compensation if applicable
4
Test and Calibrate
  • Display test pattern (color bars, grid)
  • Verify each panel receives correct portion of image
  • Adjust color calibration if needed
  • Test scaling with different source resolutions
