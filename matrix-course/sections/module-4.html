The Power of Any-to-Any Routing

In previous modules, we explored one-to-many distribution (splitters/DAs) and many-to-one switching (video switchers). Now we reach the most flexible solution: true matrix switching. With a matrix, you can route any input to any output - independently and simultaneously. This is the foundation of professional AV systems where multiple sources need to reach multiple destinations flexibly.

Matrix Architecture: The Crosspoint

A matrix switcher gets its name from the mathematical concept of a matrix - a grid where every input can connect to every output. The core technology is the crosspoint switch, an array of switches at each input-output intersection.

📐 Matrix Size Notation

8x8 Matrix: 8 inputs, 8 outputs

16x16 Matrix: 16 inputs, 16 outputs

32x32 Matrix: 32 inputs, 32 outputs

The notation "AxB" means "A inputs, B outputs". An 8x4 matrix has 8 inputs but only 4 outputs - outputs are the limiting factor.

Example: 16x8 matrix can route any of 16 sources to any of 8 displays simultaneously

🔄 Non-Blocking Architecture

Definition: All inputs can reach all outputs simultaneously

Requirement: Full crosspoint switch fabric

Implication: N² crosspoints for NxN matrix

A non-blocking matrix can connect every input to every output at the same time, with no restriction on combinations. This requires a complete crosspoint array.

Math: An 8x8 non-blocking matrix needs 64 (8×8) independent crosspoints

⚠️ Blocking vs Non-Blocking

Understanding blocking behavior is crucial for system design:

Non-Blocking Matrix

  • Full crosspoint switch fabric
  • All 16 inputs can go to all 16 outputs
  • Simultaneous routing: IN1→OUT1, IN2→OUT2, etc.
  • Higher cost (more crosspoints)
  • Required for: Mission-critical applications

Blocking Matrix (Some Models)

  • Limited switch fabric
  • Cannot route all combinations simultaneously
  • Once output is used, input may be "blocked"
  • Lower cost
  • Warning: Check specifications carefully!
Always Verify: Professional-grade matrix switchers are typically non-blocking. Budget or consumer-grade models may have blocking limitations. When in doubt, ask the manufacturer.

Audio Breakaway: Independent Audio Routing

One of the most powerful features of professional matrix switchers is audio breakaway - the ability to route video and audio independently. This means the audio from Input 1 can go to Output 3 while Video 1 goes to Output 1.

Display 1 shows: Video: Laptop 1 Audio: Document Camera Display 2 shows: Video: Laptop 2 Audio: Laptop 2 Speakers play: Audio: Laptop 1

🔊 Audio De-Embedding

Function: Extract audio from HDMI

Outputs: Analog (RCA/3.5mm), Digital (S/PDIF)

Use: Connect to external amplifiers/speakers

Matrix switchers with de-embedding can send audio from any input to dedicated audio outputs, allowing connection to separate sound systems without affecting video routing.

🎚️ Audio Embedding

Function: Insert external audio into HDMI

Inputs: Analog, Digital, Dante/network

Use: Add narration, sound system feed to video

Advanced matrices can embed external audio into the HDMI stream, replacing or mixing with the original audio. This is essential for video conferencing integration.

🔀 Audio Matrix

Function: Independent audio routing

Routing: Any audio input → Any audio output

Separation: Audio from video completely independent

Professional matrices include a complete audio matrix alongside the video matrix, allowing any audio source to reach any audio destination regardless of video routing.

EDID Management in Matrix Switchers

Managing EDID in a matrix is more complex than a simple switch because each input may need different EDID information, and each output needs to report its capabilities back to the matrix.

📋 EDID Management Modes
AUTO Mode: The matrix automatically copies the EDID from the connected display to the source. Simple but may cause conflicts when displays differ.
COPY Mode: Manually copy EDID from one output's display to specific inputs. Allows standardization across multiple sources.
PRESET Mode: Use built-in EDID presets (1080p, 4K@30Hz, 4K@60Hz, etc.). Guarantees consistency regardless of displays.
LEARN Mode: The matrix reads EDID from a connected display and saves it as a custom preset for future use.

Matrix Control Methods

Professional matrix switchers offer multiple control options for different integration scenarios:

🎛️

Front Panel

Direct button control for each input-output crosspoint. Essential for manual override and troubleshooting.

Always included
📱

IR Remote

Infrared remote for basic operation. Limited range, line-of-sight required.

Consumer/Budget models
💻

RS-232

Serial control via third-party control systems (Crestron, AMX, Extron). Reliable, deterministic.

Professional standard
🌐

TCP/IP (Web/API)

Web browser interface and REST API for modern control system integration.

Enterprise deployments
